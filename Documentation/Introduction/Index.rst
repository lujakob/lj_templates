What does it do?
================

This extension provides a basic Typo3 website setup. It is used as a starting point for your custom typo3 installation. Templating is based on the extension fluidpages and content element rendering is based on the sysext fluid_styled_content.

HTML templates, content element templates, all typoscript and assets such as css/js is outsourced into this extension. Basically all you need to do is install dependencies and this extension, set up pagetree, define baseurl and main nav entry point and a very rough website is available.
